# Drone-Apple-Digital-Twin
# Цифровой двойник БПЛА для сбора красных яблок

## Описание проекта
Разработка цифрового двойника беспилотного летательного аппарата (БПЛА) для автономного обнаружения и сбора спелых (красных) яблок. Двойник реализован в среде Unity с использованием ML-Agents для обучения интеллектуального агента методом PPO.

## Архитектура
- **Unity** — среда симуляции, физика, визуализация
- **ML-Agents** — обучение агента (алгоритм PPO)
- **Python + YOLO** — компьютерное зрение для детекции красных яблок
- **Socket-мост** — связь между Unity и Python

## Структура репозитория

├── docs/

└── README.md # Этот файл

## Ключевые особенности
- **Награды:** +5 за красное яблоко, -2 за зелёное, штраф за бездействие
- **Наблюдения:** 9 параметров (позиция, скорость, направление на цель)
- **Действия:** 4 дискретных (движение по осям + захват)
- **Детекция:** YOLOv8, обученная на 4000 синтетических кадрах
